# flySnailWebDisk

#### 介绍
自制简易网盘

#### 软件架构
c++写的简易网盘

使用开源项目
- httplib
- inja
- nlohmann

#### 编译方法

- Windows
1.  使用cmake转换vs工程

- Linux
1. `cmake ..`
2. `make`

#### 使用说明

1. templates为模板文件夹
2. ResDir为文件存储目录
3. `./FlySnailWebDisk` 启动程序
4. 默认端口为8888
